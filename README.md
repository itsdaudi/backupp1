# Daudi Kazungu - Photographer Portfolio

A responsive portfolio website for photographer Daudi Kazungu, showcasing work, journey, and contact information.

## About

This portfolio website showcases the photography work of Daudi Kazungu, a storyteller and moments collector who has been documenting life through the lens for over ten years. Inspired by Billy Miaron, Daudi captures authentic moments, emotions, and stories that preserve pieces of history.

## Features

- **Responsive Design**: Works on all device sizes (desktop, tablet, mobile)
- **Smooth Navigation**: Fixed navbar with smooth scrolling to sections
- **Interactive Gallery**: Hover effects on photo cards with image scaling
- **Timeline Section**: Visual representation of the photographer's journey
- **Contact Form**: Functional contact form for inquiries
- **Modern Styling**: Clean, professional design with subtle animations
- **Ocean Blue Theme**: Professional blue color scheme that complements photography work

## Sections

- **Home**: Hero section with name and tagline
- **About**: Biography and profile image
- **Gallery**: Collection of photographs
- **Timeline**: Journey and milestones
- **Contact**: Contact form for inquiries

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- JavaScript (for form handling)
